# 🎨 SillyTavern 美化管理器

> SillyTavern 主题美化管理扩展 —— 让你的 100+ 主题不再混乱

**作者：温水**

---

## ✨ 功能一览

- **一键切换主题** — 点击卡片即可切换，无需翻设置页面
- **导入美化** — 支持导入单个 ST 主题 JSON，美化会写入 ST 用户主题列表
- **批量导入/导出美化包** — 一次打包或导入多个美化，适合备份和社区分享
- **导入分类选择** — 导入美化时可选择导入哪些分类，也可指定导入后的本地分类
- **分类管理** — 自定义分类，把主题按风格、作者、用途归类
- **标签系统** — 每个主题可添加多个标签，搜索时更精准
- **收藏功能** — 星标你最爱的主题，收藏优先排序一键找到
- **预览截图** — 为主题添加截图，卡片展示一目了然
- **绑定背景** — 可为每个美化绑定 ST 已导入壁纸，切换美化时自动应用或取消背景
- **多种排序** — 按名称 / 最近使用 / 使用频率 / 收藏优先
- **搜索** — 支持搜索主题名、标签、作者、备注
- **网格缩放** — 主界面卡片可自由放大缩小，主题多时更容易浏览
- **批量操作** — 多选后批量分类、加标签、切换收藏
- **使用统计** — 自动记录每个主题的使用次数和最近使用时间
- **大图浏览** — Lightbox 模式浏览截图，支持左右切换
- **数据导出/导入** — 备份你的分类、标签、截图等标注数据
- **悬浮球** — 可拖拽、可自定义图片和大小的 FAB 按钮，随时打开管理器
- **明暗主题** — 管理器自身支持明/暗模式切换

## 📦 安装方法

### 方法一：通过 SillyTavern 扩展安装器

1. 打开 SillyTavern
2. 进入 **扩展** → **安装扩展**
3. 输入本仓库地址，点击安装

### 方法二：手动安装

1. 下载本仓库的 zip 文件
2. 解压到 `SillyTavern/data/default-user/extensions/` 目录下
3. 确保目录结构为：
   ```
   extensions/
   └── theme-manager/
       ├── index.js
       ├── src/
       └── manifest.json
   ```
4. 刷新 SillyTavern 页面

### 可选：安装后端存储

如需把截图、缩略图、悬浮球图片和标注数据保存到后端，请额外安装 `theme-mgr-server` 后端仓库：

1. 下载或克隆 `theme-mgr-server`
2. 将后端文件夹复制到 `SillyTavern/plugins/theme-manager/`
3. 重启 SillyTavern
4. 打开美化管理器，插件会自动优先使用后端存储；后端不可用时会回退到浏览器本地存储

## 🚀 使用说明

- **打开管理器**：点击侧栏的「美化管理」按钮，或点击右下角悬浮球
- **切换主题**：直接点击主题卡片
- **管理主题信息**：点击卡片左下角的 `⋯` 菜单按钮，可以编辑、收藏、查看截图
- **绑定背景**：在主题编辑面板中点击「绑定背景」，从 ST 已导入壁纸中选择
- **添加分类**：进入设置（底栏齿轮图标）→ 管理分类
- **批量操作**：点击底栏「多选」按钮进入批量模式
- **导入/导出美化**：进入设置 → 数据，可导入单个美化、批量导入美化或导出美化包
- **搜索**：点击顶栏搜索图标
- **排序**：点击顶栏排序图标，选择排序方式
- **刷新列表**：点击底栏刷新按钮，重新读取 ST 主题列表

## 💡 说明

- 管理器通过读取 SillyTavern 的主题列表来工作，切换美化时会调用 ST 的主题能力
- 安装 `theme-mgr-server` 后端后，分类、标签、截图、缩略图、悬浮球图片等数据会优先保存到后端；未安装后端时会回退到浏览器 IndexedDB
- 「导出标注 / 导入标注」会处理分类、标签、截图等附加数据，不修改 ST 主题文件
- 「导出美化包 / 导入美化」会读写 ST 用户主题 JSON，并尽量携带截图资源和分类标注，适合备份或社区分享
- 绑定背景使用 ST 已导入的壁纸；切换到未绑定背景的美化时，会自动恢复透明背景

## 📝 更新日志

### v3.5.5（当前版本）

- 修复柏宝库开启主题懒加载后，连续切换主题时较早的异步请求可能覆盖最后一次选择，导致多个主题看起来都变成上一套主题的问题
- 美化管理器会在切换前取得并验证完整主题，过期或失败的加载请求不会再修改当前视觉、主题名称和绑定背景

### v3.5.4

- 修复柏宝库开启主题懒加载后，重命名未加载主题可能把占位对象保存到磁盘并删除原主题的问题
- 主题改名改为保存、完整性验证、删除旧主题的安全流程，并同步刷新缓存、当前主题和背景绑定

### v3.5.3
