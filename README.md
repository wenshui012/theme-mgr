# 🎨 SillyTavern 美化管理器

> SillyTavern 主题美化管理扩展 —— 让你的 100+ 主题不再混乱

**作者：温水**

---

## ✨ 功能一览

- **一键切换主题** — 点击卡片即可切换，无需翻设置页面
- **导入美化** — 支持导入单个 ST 主题 JSON，美化会写入 ST 用户主题列表
- **批量导入/导出美化包** — 一次打包或导入多个美化，适合备份和社区分享
- **导入分类选择** — 导入美化时可选择导入哪些分类，也可指定导入后的本地分类
- **分类管理** — 自定义分类，把主题按风格、作者、用途归类
- **标签系统** — 每个主题可添加多个标签，搜索时更精准
- **收藏功能** — 星标你最爱的主题，收藏优先排序一键找到
- **预览截图** — 为主题添加截图，卡片展示一目了然
- **绑定背景** — 可为每个美化绑定 ST 已导入壁纸，切换美化时自动应用或取消背景
- **多种排序** — 按名称 / 最近使用 / 使用频率 / 收藏优先
- **搜索** — 支持搜索主题名、标签、作者、备注
- **批量操作** — 多选后批量分类、加标签、切换收藏
- **使用统计** — 自动记录每个主题的使用次数和最近使用时间
- **大图浏览** — Lightbox 模式浏览截图，支持左右切换
- **数据导出/导入** — 备份你的分类、标签、截图等标注数据
- **悬浮球** — 可拖拽、可自定义图片和大小的 FAB 按钮，随时打开管理器
- **明暗主题** — 管理器自身支持明/暗模式切换

## 📦 安装方法

### 方法一：通过 SillyTavern 扩展安装器

1. 打开 SillyTavern
2. 进入 **扩展** → **安装扩展**
3. 输入本仓库地址，点击安装

### 方法二：手动安装

1. 下载本仓库的 zip 文件
2. 解压到 `SillyTavern/data/default-user/extensions/` 目录下
3. 确保目录结构为：
   ```
   extensions/
   └── theme-manager/
       ├── index.js
       └── manifest.json
   ```
4. 刷新 SillyTavern 页面

## 🚀 使用说明

- **打开管理器**：点击侧栏的「美化管理」按钮，或点击右下角悬浮球
- **切换主题**：直接点击主题卡片
- **管理主题信息**：点击卡片左下角的 `⋯` 菜单按钮，可以编辑、收藏、查看截图
- **绑定背景**：在主题编辑面板中点击「绑定背景」，从 ST 已导入壁纸中选择
- **添加分类**：进入设置（底栏齿轮图标）→ 管理分类
- **批量操作**：点击底栏「多选」按钮进入批量模式
- **导入/导出美化**：进入设置 → 数据，可导入单个美化、批量导入美化或导出美化包
- **搜索**：点击顶栏搜索图标
- **排序**：点击顶栏排序图标，选择排序方式
- **刷新列表**：点击底栏刷新按钮，重新读取 ST 主题列表

## 💡 说明

- 管理器通过读取 SillyTavern 的主题列表来工作，切换美化时会调用 ST 的主题能力
- 分类、标签、截图、收藏等附加信息存储在浏览器的 IndexedDB 中
- 「导出标注 / 导入标注」只处理分类、标签、截图等附加数据，不修改 ST 主题文件
- 「导出美化包 / 导入美化」会读写 ST 用户主题 JSON，并附带轻量标注信息用于分类筛选
- 绑定背景使用 ST 已导入的壁纸；切换到未绑定背景的美化时，会自动恢复透明背景

## 🔧 兼容性

- SillyTavern 1.12.0+
- 在移动端和桌面端均可使用

## 📝 更新日志

### v3.0（当前版本）

这是一次大版本更新，重点补齐「美化文件本体」的导入导出、社区分享和美化绑定背景能力。

- 新增导入单个美化 JSON，导入后会写入 ST 用户主题列表
- 新增批量导入美化，支持多文件、数组格式和美化包格式
- 新增导出美化包，打包 ST 当前所有主题 JSON，并附带分类、标签、作者、备注、绑定背景等轻量标注
- 新增导入美化时选择分类：可按包内分类筛选导入，也可指定导入后的本地分类或新建分类
- 新增美化绑定背景：每个美化可绑定 ST 已导入壁纸，切换美化时自动应用
- 修复未绑定背景的美化切换后沿用上一张背景的问题，现在会自动恢复透明背景
- 新增带缩略图的背景选择弹窗，避免只看文件名无法确认壁纸
- 新增自定义悬浮球：支持上传图片/GIF/透明 PNG、调整大小、拖拽保存位置
- 优化设置面板按钮布局，导入/导出相关操作改为两行三列
- 优化多选栏位置，固定在分类栏下方、网格上方，滚动时不再被带走
- 分类管理改为拖拽排序，替代上移/下移按钮
- 导入美化时增加 `custom_css` 中 `@import` 风险提示和同名覆盖提示
- 修复导入后 ST 页面显示已应用但实际主题未生效的问题，补充直接应用主题变量的兜底逻辑
